# Mobile Robotic Manipulator

A mobile robotic manipulator project developed for the Introduction to Robotics course at the Faculty of Engineering, Alexandria University.

## 📌 Project Overview
This project aims to design and build a mobile robot equipped with an RP/PR robotic manipulator capable of performing tasks such as:
- Object sorting by color or shape
- Object arrangement
- Basic pick-and-place operations

The project combines:
- Mechanical Design
- Embedded Systems
- Control Systems
- Robotics Kinematics
- Simulation and Testing

---

## 🛠️ Technologies & Tools
- Arduino / ESP32
- Servo Motors & Sensors
- MATLAB / Simulink
- ROS (Robot Operating System)
- C++ / Embedded Programming

---

## 📂 Project Features
- Mobile robotic platform
- RP/PR manipulator design
- Forward & Inverse Kinematics simulation
- Object detection and sorting
- Real-time control system

---

## 📑 Project Requirements
- Design and build the manipulator
- Perform a robotic task using the arm
- Create a detailed technical report
- Demonstrate the project with a video presentation
